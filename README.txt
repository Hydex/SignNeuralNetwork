Instruções de uso:

Para correr a aplicação, na pasta do projeto, correr:
	java bin/out/production/SignLangNN/mieic/iart/SignLangNN/frontend/Main
A aplicação carregará uma rede já treinada.
Caso necessário um novo treino, apagar o ficheiro
	neuralnet.ser
e iniciar novamente a aplicação.

Depois do treino, é disponibilizado um menu, onde é possível testar e treinar adicionalmente a rede neuronal.
